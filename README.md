# bankapp

Project Title: Bad Bank React application

Description/Motivation: This project was to demonstrate front end skills in JavaScript and React before moving into backend and DB at the end of the course. I hosted this application in AWS.

https://petermccallumbankingapp.s3.us-east-2.amazonaws.com/index.html

Installation Guidelines: 

This project was bootstrapped with Create React App.

Inside the newly created project, you can run some built-in commands:

npm start or yarn start

Runs the app in development mode.
Open http://localhost:3000 to view it in the browser.

The page will automatically reload if you make changes to the code.
You will see the build errors and lint warnings in the console.
